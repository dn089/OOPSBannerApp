# OOPS Banner App

This project demonstrates progressive development from UC1 to UC7.

## Concepts Covered
- Java Fundamentals
- String Handling
- Arrays and Loops
- Static Methods
- Object Oriented Programming
- Encapsulation
- Abstraction
- Git Workflow

## Run Project

Compile:
javac src/OOPSBannerApp.java

Run:
java -cp src OOPSBannerApp
